# RestaurantChooser
## サービス概要
行きつけ、気になるなど飲食店の情報をたくさん登録し、その時の気分とか状況に合わせて条件（場所とかお店のジャンルなど）を絞り込み、日々のちょっとした決断を自動化します。
それ以外にもポストの投稿やいいね、ポストやお店情報の検索などSNS風の側面も持ち合わせています。
